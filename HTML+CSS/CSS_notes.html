CSS Wording Features

first declare a class for the text features, then insert the class in tag

Features Syntax
Text Position text-align
Line Height line-height
Letter Spacing letter-spacing

CSS Image Features

There is a predefined class (Method 1) for Images : "img{FEATURES}" (declared it under style)

CSS Body Features

Like Images, there is a predefined class (Method 1) for Body
One of the usage : Setting up the background

Syntax Description
background-color This would give the corresponding color to the background of the website
background-image : url(PATH) This would allow developer to make a image (local or online) as the background
background-repeat If the image is not large enough, this features would determine whether the image would be copied to cover the page
background-size This would allow the developer to adjust the size of the image or set the image to cover the page
background-attachment This would determine whether the image would follow the vertical motion of the page (roll up and down)
background-position : X-AXIS POSITON, Y-AXIS POSITION This allow feveloper to set the image position on the background

CSS Hyperlinks Features

  1. predefined groups

    2. Like image and body, a has a predefined class (Method 1) in CSS : a{FEATURES}

  2. specific attributes

    3. To specify hyperlink attributes : a:ATTRIBUTES

    Attributes Description
    hover it specifies the apperance of link when the mouse is over the link
    visited it specifies the apperance of link when users have already visited the link

CSS file

"<"link rel="stylesheet" href="STYLE_CSS_PATH"">"
